# how use 

### 1. Clone this repository

* ```
  git clone
  ```

### 2. Go repository

* ```
  cd Diemu
  ```

### 3. Install dependencies

* ```
  yarn or npm install
  ```

### 4. Execute app

* ```
  npx expo run:android
  ```
